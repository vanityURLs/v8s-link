VanityURLs

A short-link service powered by vanityURLs

Security

Default template to adapt by the instance owner.

Security model

This instance is a static-asset and Cloudflare Worker based redirect service. It should not expose a public write API, visitor accounts, cookies, or client-side analytics by default.

Infrastructure

Instance owners should use Cloudflare security controls, access protection for operational paths, safe redirect-target validation, and abuse block policies appropriate for their deployment.

Vulnerability reports

If you discover a security issue affecting this instance, report it privately to the operator. Do not publish exploit details before the operator has had time to respond.

Project security

Security issues in the vanityURLs software itself can be reported through the project's GitHub security advisory process.
